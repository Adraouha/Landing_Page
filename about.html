About the project

The Odin Project was created to provide a complete path for students to become employed as web developers while working with other students along the way. Students start with the course Introduction to Web Development, before moving on to Web Development 101 and more in-depth modules including HTML, CSS, Javascript, Ruby, Rails, Git, databases and more. The tutorials are completely flexible, allowing students to work through each lesson or jump to different lessons/projects. The Odin Project is open source and is best for beginners, or those who have tried other resources without success,who want to work as a web developer, do freelancing work, build a startup, or learn to build a website.

The Odin Project is a free coding bootcamp created in 2013 by Erik Trautman. It offers an open source curriculum to aspiring web developers from around the world, allowing them to study at their own pace. The Odin Project maintains a Discord community of nearly 718,000 students and provides job assistance after graduation.
